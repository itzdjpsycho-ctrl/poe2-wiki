## Quick Reference

### Current Top Builds (Patch 0.5)
1. **Lightning Arrow Deadeye** — #1 map clear, 34%+ of top players
2. **Twister** — untouched by nerfs, best league starter
3. **Ice Shot Deadeye / Ice Strike Martial Artist**
4. **Grenade Totems Warbringer** — easiest beginner starter
5. **Demon Recoup Infernalist**

### Key Unique Drop Sources
- **Temporalis** → Zarokh (Trial of Sekhemas 4F) + Last Flame relic
- **Hand of Wisdom and Action** → Xesht, We That Are One (Breach pinnacle)
- **Ingenuity** → King in the Mists (Ritual)
- **Rite of Passage** (Golden Charm) → Azmerian Wisp-possessed enemies
- **Olroth's Resolve** → Olroth, Origin of the Fall (Expedition)

### Defensive Layer Priority (Meta)
1. Evasion + Energy Shield (best endgame synergy)
2. Armor + Life (best for Warrior/beginners)
