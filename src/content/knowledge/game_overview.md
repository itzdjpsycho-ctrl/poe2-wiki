# Path of Exile 2 — Game Overview

## Status
- **Early Access** (as of June 2026)
- Current patch: **0.5.4 (Return of the Ancients)**
- Developer: Grinding Gear Games (GGG)
- Standalone game (not just an expansion to POE1; both games coexist)
- POE1 and POE2 share microtransactions but are separate game clients

## What's Different from POE1

| Feature | POE1 | POE2 |
|---|---|---|
| Skill gems | Socketed into gear | 9 slots on character panel |
| Crafting | Complex orb-based meta-craft | Simpler + Runeforging in 0.5 |
| Movement | Click-to-move only | Dodge roll + click-to-move |
| Boss difficulty | Scaled to speed | Slower, more deliberate; telegraphed |
| Support gems | Linked to skill in gear | Directly attached to skill gems |
| Passive respec | Regret Orbs (costly) | Gold (cheap; available Act 1) |
| Class weapon lock | Soft (ascendancy suggests) | Any class can use any weapon at endgame |
| Dual spec | Not available | Built-in dual weapon specialization |

## Core Philosophy Changes
- **Slower, more deliberate combat** — not POE1's zoom-zoom speed (though endgame gets faster)
- **Boss fights matter** — multi-phase, telegraphed attacks; can't just ignore mechanics
- **Build diversity encouraged** — any class can use any skill at endgame
- **Less punishing respec** — Gold makes experimenting with builds accessible
- **Deeper crafting** — Runeforging (0.5) adds a new deterministic crafting layer

## Game Structure
1. **Campaign:** 6 Acts (100+ bosses, 600+ monster types)
2. **Endgame:** Atlas of Worlds with Waystones (maps)
3. **Pinnacle bosses:** 7+ per patch, culminating in Arbiter of Divinity
4. **League mechanics:** Breach, Expedition, Ritual, Delirium, Abyss, Incursion + current league
