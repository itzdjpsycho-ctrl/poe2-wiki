# Path of Exile 2 Skills, Gems & Passive Tree

## Skill Gem System

Unlike POE1 (socketing gems into gear), POE2 has a **9-socket skill panel** directly on the character. No gear-dependent socket colors.

- **Skill Gems** grant active abilities: attacks, spells, minions, movement, curses, utility
- Each skill gem can hold up to **5 Support Gems** (a "6-link" equivalent)
- After upgrading a skill gem to include 5 supports = called a **6-link**
- Gems drop with quality levels; use **Gemcutter's Prisms** to improve quality
- **Uncut Gems** drop and can be engraved into chosen skills at any vendor

## Spirit Gems
- Govern persistent effects: buffs, triggered tools, minions, reservations
- Use **Spirit** resource (limited pool; expanded via gear and passive tree)
- Spirit-based minions, auras, and triggers are all Spirit Gems

## Passive Skill Tree
- **Shared tree** across all classes; classes just start in different positions
- **124 total Passive Skill Points** obtainable (leveling + quests)
- **Keystone Passives** — radically change build fundamentals, usually with downsides. Examples:
  - Hollow Palm (unarmed playstyle for Monks)
  - Others that enable poison-on-hit, elemental equilibrium, etc.
- **Respec Cost:** Use **Gold** (not orbs) to refund passive points at the Hooded One in Act 1. Fully reversible.

## Weapon Specialization
- Passive nodes can be tied to specific weapon types
- Unlocked during campaign as you level up
- Lets you run two different weapon setups with distinct passive loadouts (via Dual Specialization)

## Dual Specialization
- Two separate passive specs on one character
- Switch between them with a button press
- Enables hybrid playstyles (e.g., bow + melee weapon swap with different passive bonuses)

## Build Fundamentals
Core build checklist:
1. **Damage source**: Active skill gem + supports
2. **Passive tree path**: Route toward damage nodes + defensive nodes
3. **Defenses**: Life + one or two defensive layers (Armor, Evasion, ES)
4. **Resistances**: Must cap at 75% for Chaos, Fire, Cold, Lightning
5. **Spirit allocation**: For persistent auras/minions
6. **Gear synergy**: Stats on gear must support skill gem requirements (Str/Dex/Int)

## Support Gem Examples
- Supports drastically alter skill behavior (e.g., chain projectiles, add AOE, add damage type)
- Can create **completely different playstyles** from the same base skill

**Why:** Understanding this system is critical for building characters — it's the foundation of every build. Gem choices + support links are the most impactful decisions after ascendancy.
